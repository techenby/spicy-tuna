## Getting Started

1. Clone Repo
2. Set PHP version to 8.5
3. `composer config http-basic.composer.fluxui.dev {licence holder} {license key}`
4. `composer install`
5. `npm install`
6. `cp .env.example .env`
7. `php artisan key:generate`
8. Link the directory to Valet or Herd
